Checked shows the full URL; Unchecked shows only the hostname portion.

A string to be put between the original title and the URL (e.g., 'Title - URL' or 'Title :: URL')

 

When this is checked, the 'name' and 'id' attributes of <input /> fields will be added to the title when they have focus. This is useful if you have a page where the URL is not enough for Keepass to determine the proper auto-type. This is good for two-step logons as well. You simply create an auto-type command specific to the field, then you can determine the auto-type based on the URL and the specific field which currently has fous.
